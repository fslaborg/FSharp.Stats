fold f z a

Full Usage: fold f z a

Parameters:
    f : 'a -> float -> 'a
    z : 'a
    a : matrix

Returns: 'a

Applies a function f row wise to each element of the matrix, threading an accumulator argument through the computation. The fold function takes the second argument, and applies the function f to it and the first element of the matrix. Then, it feeds this result into the function f along with the second element, and so on. It returns the final result. If the input function is f and the elements are i0...iN, then this function computes f (... (f s i0) i1 ...) iN.

f : 'a -> float -> 'a
z : 'a
