Linker Module

Namespace: FSharp.Stats.ML.Unsupervised
Assembly: FSharp.Stats.dll
Parent Module: HierarchicalClustering

The linkage criterion determines the distance between sets of observations as a function of the pairwise distances between observations

Types

Type Description

LancWilliamsLinker

Signiture type for Lance and Williams Linker functions D(A u B,C) = alpa1 d(A,C) + alpa2 d(B,C) + beta d(A,B) + gamma |d(A,C) - d(BC))|
